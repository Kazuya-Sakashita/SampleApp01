# ”Talk helper”について

このアプリは、発語に障害を持っている方や言葉を話せない人の代わりにしゃべってくれるアプリです。

主な機能

*発語困難な方の代わりに登録した言葉を読み上げてくれます。

*登録した言葉は、他の人と共有することができます。

*他の人の言葉をbookmarkすることができます。
